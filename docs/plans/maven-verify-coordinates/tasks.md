ENFORCE caps IN THE HANDLER before any I/O (schema is advisory only): reject/truncate `len(dependencies)>100`, `suggestLimit=min(suggestLimit,10)`. Build `ResolutionContext` via `build_resolution_context(args)`. Per dependency (sequential, broad per-item try/except → `error` item on unexpected failure):
  - EXISTENCE PROBE (own loop over `_repos_for(g,a,ctx)`, `http_get` each metadata URL, do NOT use fetch_metadata's raise; http_get returns a code for ALL HTTP statuses, raises only transport): any 200 → `existenceStatus:"exists"` + `repository`=FIRST answering repo + UNION `<versions>` across ALL 200-answering repos; `"absent"` ONLY if EVERY repo returned 404; otherwise (any 401/403/429/5xx, any raised transport error, or a mix like 404+503) → `"unknown"`. `gaExists = existenceStatus=="exists"`. (Probe except: catch `URLError` AND `socket.timeout` per-repo → `unknown`-contributing, distinct from the broad per-item except that emits an `error` item.)
  - `gavExists` (if version given) = membership of version in the UNION of versions across all 200-answering repos (not first-hit). `stability` = `classify_version(latest)` ONLY if latest non-empty (omit otherwise; never classify None).
  - suggestions (only on `"absent"`): `search_maven_central(_solr_escape(artifactId), …)`; `score = max(_similarity("g:a"), _similarity("a"))` RAW; the SORT KEY applies a low-`versionCount` PENALTY (not a pure tiebreaker — a 1-version near-miss can be pushed below a high-popularity real coord even at slightly higher raw similarity) so a 1-version near-miss never outranks a high-popularity real coord; the penalty affects sort ORDER only and is NEVER folded into the emitted raw `score` or the flag; top `suggestLimit`, each `{groupId,artifactId,score,versionCount}`. Search failure → empty, no raise.
  - `likelyHallucination = (existenceStatus=="absent") and any(raw_similarity(c) >= HALLUCINATION_THRESHOLD(0.8))` over the FULL pre-truncation candidate set (NOT the truncated/de-weighted list). Never on `unknown`/`exists`. Output never asserts "safe".
